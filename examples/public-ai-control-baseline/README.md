# Public AI Control Baseline Example

This example is a compact internal decision memo built only from live public sources. It exists to validate that GPD can carry real source URLs, source IDs, claim-support metadata, fact-check findings, review guidance, and export wording through a complete paper workspace.

It also acts as a second calibration point for governance/control-paper guidance. Unlike the software supply-chain example, this memo is a smaller AI pilot control-baseline decision memo. Its artifacts show how the reusable control checks apply without turning a compact memo into a white paper: define the governed object, name the durable record, explain standards as context, state evidence currency and refresh triggers, show how evidence changes decisions, and answer the process-burden objection.

Source verification date: 2026-05-12.

Public sources used:

- NIST AI Risk Management Framework 1.0
- NIST AI RMF Generative AI Profile
- OWASP GenAI Security Project LLM01 Prompt Injection
- NCSC/CISA Guidelines for Secure AI System Development

Privacy boundary: this example is synthetic. It must not contain real person names, company names, employer names, internal titles, local paths, private drafts, or private source material.
